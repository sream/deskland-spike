# Deskland Technical Spike

This project is a lightweight technical spike for testing whether the following stack is suitable for the core Deskland experience:

* React
* TypeScript
* Vite
* Phaser

## Purpose

The goal is to validate the basic setup for a Gather-like 2D workspace experience.

This spike should stay small, focused, and easy to iterate on.

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Notes

This project uses Vite for fast local development and React for the surrounding UI layer.

Phaser is used for the interactive 2D world.
